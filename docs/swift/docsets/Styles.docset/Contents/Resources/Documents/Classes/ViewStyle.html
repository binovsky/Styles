ViewStyle

public final class ViewStyle: NSObject

The ViewStyle defines style of any UIView. Can define color, tint, border, shadow and layer properties. @see ViewStyle.Property

  • Property

    Property enumerates available options to define the ViewStyle. Available options are [.backgroundColor, .tintColor, .borderColor, .borderWidth, .cornerRadius, .opacity, .shadow]. @see Shadow
