  • TextStyle

    Define a TextStyle using TextStyle.Property array.

    Note If you set the view style on given UI element it simply takes the text assigned to it and tranforms it into NSAttributedString applying the array of TextStyle.Property.

    It can represent the style of:

    • UITextView
    • UILabel
    • UITextField

    Example
